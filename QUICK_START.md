## 🚨 Troubleshooting (2 minutes)

### **Tool Doesn't Load?**

```
1. Check browser: Chrome 90+, Firefox 88+, Safari 14+
2. Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)
3. Clear cookies: Settings → Clear browsing data
4. Try different browser
```

### **GitHub Pages Not Live?**

```
1. Check: Is repo PUBLIC? (Settings → Visibility)
2. Check: Is Pages enabled? (Settings → Pages)
3. Check: Source = main branch, / root folder
4. Wait 1 minute and refresh
5. Check Actions tab for errors
```

### **Netlify Deployment Failed?**

```
1. Rename HTML file to index.html FIRST
2. Make sure you're uploading to root folder
3. Check file permissions (should be readable)
4. Try uploading again
```
