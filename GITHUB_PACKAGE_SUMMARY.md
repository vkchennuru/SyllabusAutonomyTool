### **STEP 2: Deploy Online (5-10 min) — PICK ONE:**

#### **Option A: GitHub Pages (Recommended for Academia)**

```
1. Create GitHub account (free): github.com/signup
2. Create new repository:
   Name: SyllabusAutonomyAssistant
   Visibility: PUBLIC ← Important!
3. Upload files via GitHub.com:
   • SyllabusAutonomyAssistant.html → Rename to index.html
   • README.md
   • LICENSE
4. Enable Pages:
   Settings → Pages → Source: main / root
5. Live at: https://YOUR-USERNAME.github.io/SyllabusAutonomyAssistant/
```

**See:** [GITHUB_PAGES_SETUP.md](GITHUB_PAGES_SETUP.md) for detailed walkthrough

#### **Option B: Netlify (Easiest for Beginners)**

```
1. Go to netlify.com
2. Sign up (free)
3. Click "Deploy manually"
4. Drag & drop your files
5. Live at: https://YOUR-SITE-NAME.netlify.app
```

**See:** [DEPLOYMENT_GUIDE.md](DEPLOYMENT_GUIDE.md) for details

### **Do I need to install anything?**
No. The HTML file works in any modern browser. No installation needed.

### **Can I share this with students?**
Yes! Upload to GitHub Pages, share the link. Anyone can use it.

### **Will it cost money to host?**
No. GitHub Pages and Netlify are completely free.

### **Can I modify it?**
Yes. CC BY 4.0 allows you to modify, adapt, distribute. Just give attribution.

### **How do I update it?**
Edit index.html, push to GitHub. Site updates automatically in 30 seconds.

### **Is my data safe?**
Yes. Tool runs entirely in YOUR browser. No data sent to servers.
